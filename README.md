# UiPath-Screenplay
ScreenPlay is UiPath’s next-generation desktop agent that understands natural-language goals and navigates applications like a human. It adapts to UI changes and integrates with UiPath’s automation platform to deliver reliable, governed outcomes.

## Prerequisites
- Studio 2025.10 or later.
- UiPath.UIAutomation package, version 2025.10.20 or newer
